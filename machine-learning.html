Machine learning is a subfield of artificial intelligence, which is broadly defined as the capability of a machine to imitate intelligent human behavior. Artificial intelligence systems are used to perform complex tasks in a way that is similar to how humans solve problems.

Key concepts:

  • Data: Data is the foundation of machine learning. It can be structured or unstructured, and it's used to train and test machine learning models. Quality and quantity of data are critical.
  • Features: Features, also known as input variables or attributes, are the characteristics of the data that are used to make predictions.
  • Labels/Target:n supervised learning, data is typically divided into features and labels. The labels are the output or the value we want to predict.
